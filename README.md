# test-font

A simple project to test font integration with GitHub Pages.

## Setup

1. Add your font files to the `fonts/` folder:
   - `AlmarenaDisplay-Regular.woff`
   - `AlmarenaDisplay-Regular.woff2`

2. Push the changes to GitHub

3. Enable GitHub Pages in your repository settings (Settings > Pages > Source: main branch)

## Usage

Open `index.html` in your browser or visit your GitHub Pages URL to see the font in action.
